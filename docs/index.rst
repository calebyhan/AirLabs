Introduction
------------
This library provides a pure Python interface for the `AirLabs API <https://airlabs.co/docs/>`_.

Usage
------------
To use this library, first create an `api key <https://airlabs.co/account>`_ and run::

    import airlabs
    api = airlabs.API("api key here")

Then, the endpoints may be used.
